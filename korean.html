Introduction

프랑스 출신 데이터 분석가로서 서울에 거주 중입니다. 프랑스어, 영어, 한국어, 스페인어를 능숙하게 구사하는
경영 학과 전공자입니다. 인턴십과 창업 프로젝트를 통해 마케팅, 영업, 콘텐츠 제작 등 다양한 경험을 쌓았습니다.
또한 Python, SQL, Tableau, Excel, HTML 및 CSS에서 능숙합니다.
현재는 저의 많은 관심 분야에 대해서 데이터 분석을 수행하고 결과물을 게시물로 발표합니다.

이 페이지에서는 제 모든 프로젝트를 확인할 수 있습니다.
각 프로젝트의 "프로젝트 보기"를 클릭하여 해당 프로젝트의 GitHub 페이지로 이동할 수 있습니다.
또한 각 GitHub 페이지에서 블로그 페이지 또는 Tableau 버튼을 찾아 해당 게시물이나 데이터 시각화를 확인할 수 있습니다.
